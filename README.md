# RL-agents

A collection of beginner-to-intermediate reinforcement learning agents implemented in Python notebooks.

## Contents

- RL Bandit agent
- RL GridWorld value-iteration agent
- RL TD(0) prediction agent
- RL Monte Carlo Blackjack agent
- RL SARSA agent
- RL Q-learning agent
- RL agent FrozenLake
- RL CartPole discretized Q-learning agent
- RL DQN agent
- Agent-openAI-API need

## Learning path

This repo follows a classical reinforcement learning progression:

1. Bandits
2. GridWorld / Value Iteration
3. TD(0)
4. Monte Carlo
5. SARSA
6. Q-learning
7. FrozenLake
8. CartPole with discretization
9. DQN
